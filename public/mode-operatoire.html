01 — Introduction

PRS Simulator — Mode Opératoire

Ce simulateur reproduit le fonctionnement d'un poste PRS (Poste à Relais Statiques). Il permet au formateur de construire une topologie de voie, de configurer un pupitre d'exploitation, et à l'apprenant de pratiquer la formation et la destruction d'itinéraires.

02 — Construction

Construire la topologie TCO

La construction suit un ordre logique : nœuds → tronçons → zones CDV → signaux → aiguilles.

Éléments disponibles

Catalogue des objets TCO
1
Placer les nœuds Z
Chaque nœud représente une jonction de voie (entrée/sortie de zone, point de branchement). Cliquez sur le canvas pour les placer. Glissez le nom pour le repositionner.
2
Tracer les tronçons C
Cliquez sur le nœud de départ, puis sur le nœud d'arrivée. Le tronçon est automatiquement courbé. Une fois sélectionné, tirez la poignée bleue pour ajuster la courbure.
3
Définir les zones CDV W
Passez en mode Zone CDV, sélectionnez une zone dans le panneau Propriétés (ou créez-en une), puis cliquez sur les tronçons à inclure. Chaque zone prend automatiquement une couleur distincte.
4
Placer les signaux S
Cliquez sur un tronçon pour y déposer un signal. Dans le panneau Propriétés : ajustez la position (slider 0–100%), le sens de circulation (A→B ou B→A), et configurez les zones d'approche (ZAp / EAp).
5
Placer les aiguilles A
Cliquez sur un nœud existant pour y attacher une aiguille. Dans les Propriétés : désignez le tronçon d'entrée, la branche directe, la branche déviée. L'aiguille est initialement en position directe (straight).
Pour créer des nœuds cachés (jonctions sans balise visuelle), cochez "Masqué" dans le panneau Propriétés. Utile pour les entrées/sorties de zone hors TCO visible.

04 — Itinéraires

Formation d'un itinéraire

La formation d'un itinéraire se fait en deux temps : appui bouton → vérification → activation automatique après 2,5 s.

États successifs du bouton et du TCO

Cas particuliers

Conflit d'itinéraire

Le panneau propriétés du pupitre liste les raisons du conflit (ZP, ZEA, CIA, itinéraire incompatible).

Itinéraire en transit progressif

Le transit se libère zone par zone au passage du train. La zone précédente reprend sa couleur normale.

05 — Destruction

Destruction d'itinéraire — Procédure DMT

Quand un E.Ap (Espacement d'approche) est actif, la destruction normale est bloquée. La procédure DMT (Destruction Manuelle Temporisée) est requise.

1
Poser la FC (Fermeture de Contrôle)
Cliquer sur le bouton FC associé au signal d'entrée de l'itinéraire. Le bouton passe au rouge → le signal se ferme immédiatement.
2
1er geste — Appuyer sur le bouton itinéraire
Le bouton passe en état ENREG.. Le délai réglementaire (temps de constatation d'arrêt) démarre.
3
Attendre l'expiration du délai
Le compte à rebours s'affiche dans le panneau DMT du pupitre. Le bouton clignote en orange. Le 2e geste est bloqué jusqu'à 0:00.
4
2e geste — Appuyer à nouveau sur le bouton
Annulation E.Ap, libération des aiguilles, destruction de l'itinéraire. Le bouton repasse à l'état IDLE.
Sans FC préalable, le 1er geste est inopérant et le panneau DMT l'indique clairement.
06 — FC / Annulateurs

Fermetures de Contrôle et Annulateurs

Fermeture de Contrôle (FC)

La FC active empêche l'ouverture du signal même si un itinéraire valide l'utilise. Indispensable pour la procédure DMT.

Annulateur de zone

L'annulateur neutralise la détection d'occupation pour les zones configurées. Utile en cas de fausse occupation.

07 — Simulation

Simulation d'un train

États du train et rendu TCO
Un train placé en mode Train X hérite automatiquement du sens défini par l'itinéraire actif sur le tronçon. Sa vitesse est réglable dans le panneau Simulation en bas du canvas.
08 — Diagnostic

Boutons de Diagnostic

Les boutons de diagnostic sont dans la section DIAGNOSTIC du pupitre. Ils fonctionnent en maintien : actifs tant que le clic est enfoncé.

Test Zone

Avant / Pendant le test

Toutes les zones libres (sans train, sans occupation manuelle, sans dérangement) s'illuminent en jaune. Relâcher remet l'affichage en état normal. Zones occupées et en dérangement sont exclues.

Test Aiguille

Vue apprenant — branches cachées

En vue apprenant, les branches sont masquées par défaut. Le Test Aiguille révèle uniquement la branche dont le contrôle électrique est confirmé. En cas de discordance : aucune branche n'apparaît.

Injecter une discordance (vue formateur)

Sélectionnez une aiguille sur le canvas. Dans le panneau Propriétés, cochez Discordance branche directe ou Discordance branche déviée pour simuler une perte de contrôle de position.

09 — Apprenant

Vue Apprenant

La vue apprenant est accessible via le bouton Vue apprenant dans la barre d'outils. Elle partage le même état que la vue formateur en temps réel.

Mode Arrangement

Le bouton ⠿ Arranger permet à l'apprenant de repositionner et redimensionner librement les boutons du pupitre. Il peut également ajouter des plaques de repérage (libellés blancs sur fond noir).

Ce repositionnement est propre à la session et ne modifie pas la configuration formateur.
